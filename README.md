# stack-gas
Füstgáz emisszióját kiszámoló Java program. 
Az adatgyűjtő által összegyűjtött nyeres adatok feldolgozása, korrekciója, és a végleges egyperces és félórásértékek mentése.

## Built with

 - Java SE JDK 1.8
 - Maven 4.0
 - MySQL 10.13
 - Java Swing
 - Apache NetBeans IDE 8.1

Version: 3.17.0
